# Novixel Labs Archive

This folder holds older docs, assets, and experiments that are useful for reference but should not define the current public site.

Current public positioning is handled by the main tabbed homepage:

- Home
- Services
- Projects
- Products
- About
- Contact

Archived material can still be mined for future copy, screenshots, or product history, but it should be reviewed before being linked from the live site.

## Folders

- `legacy-docs/`: setup notes, older strategy docs, investor material, and internal planning files.
- `legacy-assets/`: unused root-level images, logo variants, and old CSS/JS files moved out of the live root.
- `local-private/`: ignored local-only artifacts that are not meant to be published.

## Cleanup Rule

If an archived file is needed again, copy the useful part into an active page or doc and update the wording to match Novixel Labs before linking it publicly.
