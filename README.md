# Novixel Labs Website

Source code for [novixel.ca](https://novixel.ca), the public site for **Novixel Labs**.

## Positioning

**Automation Engineering & AI Systems**

Novixel Labs builds workflow automation, AI-powered tools, SaaS systems, dashboards, APIs, and custom operational software for remote-first clients. Select premium technical support remains available locally in Dawson Creek, BC.

## Main Experience

The primary site is a tabbed interface in `index.html`:

- **Home**: brand identity, proof, featured systems, and free Workflow Call entry point
- **Services**: workflow automation, AI systems, custom software, and selective local support
- **Projects**: GlassTracker, NovaLite, LocalLarry, Command Center, and labs work
- **Products**: available products plus clearly labeled labs and archived work
- **About**: company direction and operating principles
- **Contact**: structured workflow inquiry intake for free calls, paid audits, and build requests

## Audit Model

- **Free Workflow Call**: quick intro call to understand the problem and see whether automation is worth pursuing.
- **Paid Workflow Audit**: deeper review where Novixel Labs maps the process, identifies automation opportunities, and delivers a written roadmap.
- **Build Phase**: quoted software or automation work after the audit defines the practical build plan.

## Supporting Pages

- `computer-solutions.html`: premium local technical support page
- `locallarry/`: LocalLarry product page
- `atomic/`: Atomic Diff product page
- `products/ai-starter-kit.html`: starter kit product page
- `checkout-success/`: customer download completion flow

Legacy URLs such as `Novixel.html`, `about.html`, and `products.html` route visitors into the matching homepage tabs.

## Stack

This is a static GitHub Pages site using:

- HTML, CSS, and vanilla JavaScript
- `style.css` for the shared dark lab-inspired design system
- `script.js` for theme handling, tabs, product filtering, and structured mail inquiries
- Stripe-hosted payment links and the existing download-completion flow for products

## Design Reference

See `DESIGN.md` for the Novixel Labs color codes, theme tokens, typography, layout patterns, and component styling rules.

## Local Preview

Open `index.html` directly in a browser, or serve the directory with a lightweight local static server.

## Deployment

The site is served through GitHub Pages from the `main` branch with the custom domain configured in `CNAME`.
